AAPP Manager • Documentación & Guía

Referencia rápida de secciones, datos esperados y flujos de uso.

Contexto

Cómo funciona el sistema

Persistencia local

Toda la base se guarda en tu navegador mediante localStorage. Podés exportar o importar el estado completo (Excel o JSON) desde la opción Datos en el topbar. Si el almacenamiento está bloqueado, la app arranca con un seed mínimo en memoria.

Datos relacionados

Las entidades están conectadas: planes y extras pertenecen a una empresa, campañas se vinculan a un SaaS y los clientes referencian su plan, extras contratados y dominios asignados. Los contadores del dashboard y los listados usan esas relaciones para resumir la actividad.

Operaciones rápidas

El buscador global filtra por nombre, URL o notas. Desde el topbar podés cambiar de tema, abrir herramientas de datos o exportar todo a Excel. La sección POS registra ventas rápidas y la pestaña Balance resume ingresos vs. egresos.

Sidebar

Pestañas y su propósito

Usá el menú lateral para saltar a cada sección de la guía.

Dashboard

Punto de partida con KPIs rápidos: cantidad de SaaS cargadas, clientes activos, campañas y servicios extra contratados. Incluye indicadores de salud (ventas POS, egresos) y resúmenes por empresa para validar crecimiento y gastos.

  • Panel de KPIs instantáneos y tablas de salud (ventas vs. egresos, campañas recientes).
  • Resumen de planes, extras y dominios por empresa para entender cobertura.
  • Badges de estado que reaccionan al filtro de búsqueda global.

Empresas

Listado central de las empresas SaaS que gestionás. Cada registro incluye logo, sitio principal y enlaces de registro y login. Los contadores de planes y clientes permiten ver el impacto de cada SaaS de un vistazo.

  • Acceso a crear, editar o borrar empresas sin afectar el resto de datos.
  • Contadores automáticos de clientes y planes relacionados.
  • Enlaces directos para visitar el sitio, el registro o el login.

Dominios

Seguimiento de dominios operativos por empresa, con opción de asociarlos a un cliente. También se documenta proveedor, estado y notas para saber dónde y cómo gestionarlos.

  • Filtro rápido por empresa para revisar dominios de un SaaS específico.
  • Campos para proveedor, estado (activo/suspendido) y notas de seguimiento.
  • Botones de edición y borrado para mantener el inventario actualizado.

Planes

Definí los planes ofrecidos por cada SaaS, con precios, frecuencia, descripción y features incluidas. Estos planes se usan luego en clientes, revendedores y el flujo POS.

  • Frecuencia configurable (mensual, anual, etc.) y precios numéricos.
  • Listado de features fijas y variables para detallar alcance.
  • Integración directa con clientes y revendedores.

Campañas

Registro de campañas de adquisición o activación por empresa, con gastos diarios y totales, alcance, vistas y costo por conversación para evaluar la eficiencia publicitaria.

  • Captura de gasto diario y total más métricas de performance.
  • Notas libres para hipótesis, plataformas usadas o próximos pasos.
  • Relación directa con el SaaS para consolidar costos en el balance.

Clientes

Base de clientes con credenciales y notas. Cada cliente referencia su SaaS, el plan contratado, los extras activos y los dominios asignados.

  • Campos para email, contraseña (opcional) y enlaces útiles.
  • Selección múltiple de extras y asociación con dominios existentes.
  • Fecha de alta para trazar cronologías y renovaciones.

Extras

Servicios adicionales ofrecidos por cada empresa (ej. soporte premium, integraciones). Admiten features fijas y variables, frecuencia de cobro y precio.

  • Frecuencia única o recurrente para reflejar cobros puntuales.
  • Compatibles con el flujo POS y con selección múltiple en clientes.
  • Visibilidad por empresa para evitar mezclar catálogos.

Precios Revendedor

Configurá precios de reventa tomando como fuente un plan o servicio extra. Se detallan costos, precio de venta, tiempos de entrega y requisitos.

  • Selector de origen (plan o extra) para calcular margen.
  • Campos de requisitos para compartir con el partner.
  • Generación de página HTML de precios para compartir por WhatsApp o email.

Partners

Agenda de partners y afiliados con datos de contacto, compañía y comisión pactada. Permite centralizar quién vende o implementa cada SaaS.

  • Campos para email, teléfono y notas de acuerdos.
  • Referencia de comisión para cálculos de payout.
  • Útil para coordinar con revendedores y campañas conjuntas.

POS

Punto de venta rápido para registrar nuevas ventas: seleccioná empresa, plan, extras, método de pago y notas. Cada registro alimenta los contadores y el balance.

  • Captura de comprador (nombre y email) y fecha de compra.
  • Compatibilidad con extras múltiples y campañas vinculadas.
  • Botón dedicado para generar ventas sin pasar por el CRUD de clientes.

Balance

Resumen financiero simple que cruza ventas POS contra egresos manuales. Permite ver saldo, registrar egresos y revisar el historial.

  • Formulario corto para cargar egresos con nombre, monto y fecha.
  • Saldo calculado combinando ingresos POS y egresos.
  • Tabla histórica con opción de borrar movimientos erróneos.

Ayuda

Recordatorios y tutoriales cortos integrados en la app principal: incluye ejemplos de WhatsApp, estructura de plantillas de email y tips para documentar accesos y links.

  • Snippets listos para compartir con clientes o revendedores.
  • Ejemplos de estructura de mensajes para entregar dominios, accesos y facturación.
  • Enlaces rápidos para volver al dashboard y seguir operando.
